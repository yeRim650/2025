## Conditional Rendering
* 조건에 따른 렌더링
* JavaScrip의 Truthy와 Falsy
* Truthy true는 아니지만 true로 여겨지는 값
* Falsy false는 아니지만 false로 여겨지는 값
### Element Variables
* Element를 변수처럼 저장해서 사용
### Inline Conditions
* 조건문을 코드 안에 집어넣는 것
### InIne if-Else
* If-Else문의 경우 ? 연산자를 사용
### Component 렌더링 막기
* null을 리턴하면 렌더링되지 않음
